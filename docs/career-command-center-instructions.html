How to use

Career Command Center

An AI-powered career operating system: paste a job posting and Claude turns it into a tracked application, then a telemetry-style dashboard, a drag-and-drop pipeline, and a coaching library keep the whole search moving.

  • Next.js
  • TypeScript
  • Claude API
  • Prisma
  • Supabase

Explore the public demo

The hosted demo is read-only and loaded with a realistic sample pipeline of fifteen applications — you can browse everything, but writes are politely declined server-side, so feel free to click anything.

  1. Start on the Dashboard

    The landing page answers "what should I do next?" at a glance: a pipeline funnel, response/interview/offer rates, this week vs. last week, today's focus list, applications that need attention (stale or closing soon), upcoming interviews, and a live activity feed.

  2. Open the Kanban board

    Ten stages from Interested to Offer Extended. Each card shows salary, location, priority, days in stage, and the next action. Applications idle for 14+ days get an amber stale badge automatically. Try dragging a card — the demo will show you the read-only notice, which is itself the feature working.

  3. Click into any application

    Every application is a living workspace: an AI summary of the role, an automatic timeline of every change, plus tabs for notes, interviews, contacts, the archived job description, and documents. Nothing is entered twice — moving a card writes the timeline for you.

  4. Browse Resources

    A built-in library of 29 original career guides across 16 categories — resumes, ATS systems, behavioral interviews, salary negotiation — plus a prompt library with one-click-copy prompts for Claude, ChatGPT, Gemini, Perplexity, and NotebookLM.

  5. Press ⌘K anywhere

    The command palette searches every application and jumps to any page instantly.

What the full version adds

  1. AI job intake

    Paste a job posting URL (or the posting text) and Claude extracts ~23 structured fields — company, role, salary range, remote status, skills, benefits, recruiter, closing date — into an editable review screen. One confirmation click and the application is tracked, summarized, and on the board. Sites that block scraping fall back to paste automatically; a manual form always works.

  2. AI summaries and insights

    Each saved application gets a role summary, key requirements, and ATS-relevant keywords. The dashboard's insight card reads your live pipeline stats and gives one specific, actionable observation — like a coach reviewing your funnel.

  3. Your data stays yours

    The full version runs locally with a private database and your own Anthropic API key — applications, notes, and documents never leave your machine except the AI calls. Everything in the demo works identically, just writable.
