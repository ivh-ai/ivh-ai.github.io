How to use

Formula One Explorer

One app for the whole sport: an interactive course that takes you from "what's a pit stop?" to fluent fan, live timing during race weekends, and every season since 1950 — searchable from a single ⌘K palette.

  • Next.js
  • TypeScript
  • Live F1 data
  • 96 unit tests

Run it locally

  1. Clone and start — no keys needed

    Clone the repo, then npm install and npm run dev. That's the whole setup: the app runs entirely on free public APIs (Jolpica, OpenF1, Open-Meteo), so there are no accounts to create and no environment variables to configure.

  2. Open localhost:3000

    The homepage greets you with a countdown to the next Grand Prix, the current championship leaders, and rotating F1 facts — all fetched live.

If you're new to F1

  1. Start with the beginner's path

    Head to Learn F1. The course has 22 lessons across five tracks — foundations, machinery, race craft, rules, and deep dives — and a suggested nine-lesson beginner's path that takes you from zero to race-ready. Each lesson has hand-drawn diagrams, key takeaways, myth-busting, and fun facts.

  2. Quiz yourself, flip the flashcards

    Every lesson ends with an interactive quiz (with explanations for every answer) and a deck of flip-card flashcards. Mark lessons complete as you go — progress is saved on your device.

  3. Hover anything you don't know

    The Glossary holds 100+ terms — undercut, dirty air, parc fermé — searchable and filterable by category. The Rules page explains the sporting, technical and financial regulations in plain language.

If you're already a fan

  1. Watch sessions in the Live Race Center

    During any session, Live shows the full timing screen: positions, gaps, intervals, last/best laps, tire compounds and age, pit stops, race-control flag messages, trackside weather, and team radio — refreshing every four seconds. Between sessions it replays the most recent one, clearly labeled.

  2. Dive into 75+ years of archive

    The Seasons archive covers every championship since 1950 — full calendars, race and qualifying results, standings, and editorial notes on the landmark years. Driver and team pages compute career statistics from complete race history.

  3. Settle debates with the Compare tool

    Pick any two drivers in history — Senna vs. Verstappen, Fangio vs. Hamilton — and get head-to-head bars plus a radar chart normalized by per-start rates, so short brilliant careers compare fairly with long ones.

  4. Play strategist in the Simulators

    Run championship permutations ("who can still win?"), duel two tire strategies against a degradation model, or test your reactions in the pit-stop challenge — the record crew did it in 1.80 seconds.

  5. Press ⌘K for anything

    The global command palette searches drivers, teams, circuits, lessons, glossary terms, rules and all 75+ seasons instantly.
