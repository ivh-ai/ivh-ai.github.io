About

MathSprint was created by Ishaan Hattangady using Claude Code — Anthropic's AI coding tool — without writing a single line of code by hand.

Inspired by fast-paced quiz games, Ishaan wanted to build a clean, distraction-free tool for sharpening mental arithmetic — combining a smooth interface with the challenge of answering as many math questions as possible in 90 seconds.
